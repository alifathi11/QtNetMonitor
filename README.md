# QtNetMonitor

QtNetMonitor is a small client–server project built with **Qt 6**, **QML**, and **C++**

The project demonstrates how a Qt/QML-based client communicates with a TCP server.

---

## Features

### Client (Qt / QML)
- Qt 6 + QML user interface
- CMake-based build system
- TCP socket communication
- C++ ↔ QML integration 

### Test Server
- Simple TCP server for client testing
